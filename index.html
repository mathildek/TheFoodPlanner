Explication Projet

Notre projet permet à son utilisateur (particulier, hôtel ou restaurant) de limiter le gachis en achetant uniquement ce qu'il a besoin au point de vu des aliments nécessaires.

  • MATIN
  • MIDI
  • SOIR
  • Liste

Explication Code

Le programme se décompose en plusieurs Userformes qui permettent de controller les repas, ajouter des plats personnalisés dans la base de données et de créer la liste de course.
Quand il ajoute ses plats préférés, l'utilisateur peut ajouter le nombre d'ingrédients souhaités pour que la liste de course soit le plus précise.
On utilise une feuille Excel jouant le rôle de base de données.

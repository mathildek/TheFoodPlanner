Explication Projet

Notre projet permet à son utilisateur (particulier, hôtel ou restaurant) d'organiser les repas prévus pour une semaine. Pour chaque jour, l'utilisateur choisit la composition de ses repas et peut par la suite, imprimer la liste de course associée. Le programme comporte déjà une petite base de donnée avec des plâts communs mais l'utilisateur peut ajouter ses plats préférés et personnaliser ainsi la base de données.
L'objectif de ce programme est d'optimiser le temps d'organisation des particuliers, hôtels ou restaurants en supprimant les tâches répétitives. Cela permet également d'éviter le gâchis en achetant uniquement ce dont on a besoin.

Explication Code

Le programme se décompose en plusieurs Userformes qui permettent de controller les repas, ajouter des plats personnalisés dans la base de données et de créer la liste de course.
Quand il ajoute ses plats préférés, l'utilisateur peut ajouter le nombre d'ingrédients souhaités pour que la liste de course soit le plus précise.
On utilise une feuille Excel jouant le rôle de base de données.

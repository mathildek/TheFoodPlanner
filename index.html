Explication Projet

Notre projet permet à son utilisateur (particulier, hôtel ou restaurant) d'organiser les repas prévus pour une semaine. Pour chaque jour, l'utilisateur choisit la composition de ses repas et peut par la suite, imprimer la liste de course associée. Le programme comporte déjà une petite base de donnée avec des plats communs mais l'utilisateur peut ajouter ses plats préférés et personnaliser ainsi la base de données.
L'objectif de ce programme est d'optimiser le temps d'organisation des particuliers, hôtels ou restaurants en supprimant les tâches répétitives. Cela permet également d'éviter le gâchis en achetant les quantités nécessaires.

  • MATIN
  • MIDI
  • SOIR
  • Ajouter un plat
  • Liste

Explication Code

Le programme se décompose en 5 Userforms. Trois permettent de choisir les 3 repas quotidiens. On utilise une feuille Excel jouant le rôle de base de données.
Un quatrième Userform permet d'ajouter un plat personnalisé et ses ingrédients dans la base de données. Un dernier Userform permet d'établir automatiquement la liste de courses associée aux plats de la semaine et adaptée au nombre de personnes.
